Render.Backend.Atlas

Description

Módulo de leitura e processamento de Texture Atlas, permitindo que várias texturas se encontrem num único ficheiro de imagem, permitindo um uso mais eficiente da memória.

Um Texture Atlas necessita de ser uma imagem de 24-bits para evitar a utilização de um offset para as coordenadas.
